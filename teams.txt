Локально:
На диске E создать папку the lessons, открыть папку и правой кнопкой мышки выбрать Git Bash Here > открывается консоль для работы с Git.

Последовательно выполняем команды на Git Bash:

git init  созд. локальный репозиторий > the lessons (master)

git config --global user.name "VeronikaSafroshkina"  настройка имени пользователя, кот. удаленно будет делать коммиты в этом репозитории

git config --global user.email "veronika.s@bk.ru"  настройка эл.почты пользователя, кот.удаленно будет делать коммиты в этом репозитории

git config --list  проверка конфигурационных данных, см.строки user.name и user.email

git status показ. все файлы репоз-я, готовых к индексированию Untracked files(красные)

git add .  добавить файлы

git status зеленые файлы под присмотром Git, но пока без ревизии

git commit -a -m"init" заиндексировали файлы

ПК - последовательно выполняем команды на Git Bash:

git remote add origin https://github.com/VeronikaSafroshkina/tasks.git указали адрес удаленного репозитория

git remote показывает сколько репозиториев в удаленном доступе и короткое наименование origin

git remote -v показывает ссылки откуда (fetch) копировать с удаленного репозитория и куда (push) передавать изменения на удаленный репозиторий

git push -u origin master протолкнуть данные с локального репозитория на удаленный
